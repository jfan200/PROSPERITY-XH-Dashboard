# 移动端优化页面

## 访问方式

启动服务器后，通过以下 URL 访问移动端页面：

```
http://localhost:3001/mobile.html
```

## 功能特性

### 📱 针对手机屏幕优化
- 移除侧边栏，使用底部导航栏
- 卡片式布局，触摸友好
- 大字体、大按钮，便于点击
- 支持 iOS/Android 安全区域

### 🏠 首页 (Dashboard)
- **KPI 卡片** - 销售额、订单数、客单价、高峰时段
- **每小时销售图表** - 支持柱状/折线切换
- **本周概览** - 周累计、营业天数、最佳营业日
- **最近订单** - 点击查看详情
- **热销菜品** - TOP 5 排行

### 📋 订单页
- 日期选择器（左右切换、回到今天）
- 搜索功能（订单号、金额、类型）
- 订单列表（卡片式展示）
- 点击查看订单详情

### 📊 菜品页
- 热销排行榜
- 商品分析数据

### 📷 摄像头页
- 摄像头实时画面
- 状态显示

## 交互特性

### 🔄 同步数据
- 点击右上角同步按钮
- 显示同步状态横幅
- 成功/失败提示

### 🌓 主题切换
- 深色/浅色主题
- 自动保存偏好
- 图表跟随主题

### 📄 订单详情
- 全屏模态框
- 菜品明细列表
- 支付汇总
- 打印功能

## 技术栈

- **HTML5** - 语义化标签
- **CSS3** - CSS 变量、Flexbox、Grid
- **JavaScript** - 原生 ES6+
- **Chart.js** - 图表渲染

## 文件结构

```
mobile.html    # 移动端 HTML 页面
mobile.css     # 移动端样式
mobile.js      # 移动端逻辑
```

## 响应式设计

- 默认适配手机屏幕（< 768px）
- 平板/桌面端自动居中显示（max-width: 480px）
- 支持横屏/竖屏切换

## 注意事项

1. 移动端页面与桌面端共用同一个 API 后端
2. 数据同步状态在两个页面间共享
3. 建议在手机浏览器中添加到主屏幕以获得最佳体验
